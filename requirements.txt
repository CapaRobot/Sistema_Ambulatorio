======================================================================
         GUÍA RÁPIDA - CÓMO EJECUTAR EL SISTEMA BIO-LAB
======================================================================

Abra la terminal en la carpeta del proyecto, copie el bloque de 
comandos según su sistema operativo, péguelo y presione ENTER:

----------------------------------------------------------------------
PASO 1: INSTALACIÓN AUTOMÁTICA (Elija su sistema)
----------------------------------------------------------------------

* SI USTED USA WINDOWS:
  Copie y pegue estas 3 líneas juntas en su terminal:

python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt


* SI USTED USA LINUX o MAC:
  Copie y pegue estas 3 líneas juntas en su terminal:

python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

----------------------------------------------------------------------
PASO 2: ENCENDER EL SISTEMA
----------------------------------------------------------------------
Una vez que termine la instalación, ejecute este último comando:

python backend/app.py

----------------------------------------------------------------------
PASO 3: ABRIR EN EL NAVEGADOR
----------------------------------------------------------------------
Abra su navegador de internet (Chrome, Firefox, etc.) e ingrese a:

http://127.0.0.1:5000/
